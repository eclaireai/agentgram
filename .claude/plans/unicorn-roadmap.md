# Agentgram Unicorn Roadmap

## Phase 1: NOW (cold-start killer)
- [ ] PR→Recipe Reverse Extractor — mine GitHub repos for recipes
- [ ] Codebase Fingerprinting — scan project before recipe matching
- [ ] Provenance-attached recipes — add causal graph to recipe format

## Phase 2: After 100 users
- [ ] Fingerprint-based recipe matching (not just keyword search)
- [ ] Recipe composition (combine small recipes into workflows)
- [ ] Cross-session memory graph

## Phase 3: After 1,000 executions
- [ ] Recipe Genomes — evolutionary mutation from execution deviations
- [ ] Anti-pattern recipes (what NOT to do, extracted from failures)
- [ ] Aggregate codebase health scores

## Phase 4: After 10,000 users
- [ ] Recipe Economics — creator earnings from cost savings
- [ ] Agent-to-agent teaching via provenance transfer
- [ ] Recipe pricing / marketplace revenue model

## Key Insight
Every merged PR is a recipe that already happened. Mining git history
solves cold-start instantly. 1,000 repos → 50,000 recipes → instant
critical mass → network effects kick in.
