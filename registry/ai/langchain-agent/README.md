# LangChain Agent with Tools

> Build an AI agent that can search, browse, and call your APIs autonomously

## Why This Exists

A single LLM call answers questions. An agent solves problems. The difference is tools and a reasoning loop. This recipe gives you both without having to understand the internals.

## Steps (6 total)

1. ▶️ **Install LangChain and OpenAI integration**
   `npm install langchain @langchain/openai @langchain/community`

2. 📄 **Create web search tool wrapping Tavily or SerpAPI with result formatting**
   `lib/tools/search.ts`

3. 📄 **Create database query tool with safe SQL execution and row limit**
   `lib/tools/database.ts`

4. 📄 **Create createAgent() with tools, memory (ConversationSummaryMemory), and streaming executor**
   `lib/agent.ts`

5. 📄 **Create agent API route with session management and streaming response**
   `app/api/agent/route.ts`

6. ✏️ **Add LANGCHAIN_API_KEY, LANGCHAIN_TRACING_V2=true, TAVILY_API_KEY**
   `.env.local`
