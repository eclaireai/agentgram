# AI Image Generation

> Generate, edit, and vary images with DALL-E 3 and Replicate Flux

## Why This Exists

Image generation is the feature users show their friends. It drives virality. This recipe handles the full pipeline: generate → store → serve — not just the API call.

## Steps (6 total)

1. ▶️ **Install OpenAI, Replicate, and S3 clients**
   `npm install openai replicate @aws-sdk/client-s3`

2. 📄 **Create generateImage(prompt, model) — supports DALL-E 3 and Replicate Flux, returns URL**
   `lib/image-gen.ts`

3. 📄 **Create storeImage(url) that fetches the generated image and uploads to R2/S3 with unique key**
   `lib/image-storage.ts`

4. 📄 **Create POST /api/generate with rate limiting, prompt moderation, generate, store, return URL**
   `app/api/generate/route.ts`

5. 📄 **Create gallery with masonry layout, download button, and share link**
   `components/ImageGallery.tsx`

6. ✏️ **Add OPENAI_API_KEY, REPLICATE_API_TOKEN, R2_BUCKET_URL, and R2_ACCESS_KEY**
   `.env.local`
