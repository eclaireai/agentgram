# AI Structured Output (Tool Calls)

> Make AI return typed JSON every time — no more parsing hallucinated strings

## Why This Exists

AI that returns unstructured text is a prototype. AI that returns typed, validated JSON is a product. Structured output is the bridge from "demo" to "shipped."

## Steps (6 total)

1. ▶️ **Install Vercel AI SDK, OpenAI provider, and Zod**

2. 📄 **Define Zod schemas for your AI outputs (e.g., ProductSchema, SummarySchema)**

   `lib/ai-extract.ts`

4. 📄 **Create streamExtract<T>(prompt, schema) using streamObject() for progressive UI updates**
   `lib/ai-stream-object.ts`

5. 📄 **Create extraction API route with input validation and error handling**
   `app/api/extract/route.ts`

6. 📄 **Add tests with mocked AI responses verifying schema validation**
   `tests/ai-extract.test.ts`
