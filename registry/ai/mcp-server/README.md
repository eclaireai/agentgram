# MCP Server (Model Context Protocol)

> Give Claude and GPT-4 direct access to your database, APIs, and files

## Why This Exists

MCP is the USB-C of AI — a standard way to connect any AI to any data source. Instead of copy-pasting context into prompts, your AI agent calls your tools directly. This is the future of AI-native apps.

## Steps (7 total)

1. ▶️ **Install MCP SDK and Zod for schema validation**
   `npm install @modelcontextprotocol/sdk zod`

2. 📄 **Create MCP Server with name, version, and capabilities declaration**
   `mcp/server.ts`

3. 📄 **Register tools with name, description, inputSchema (Zod), and handler function**
   `mcp/tools/index.ts`

4. 📄 **Register resources (database tables, file contents) as readable context**
   `mcp/resources/index.ts`

5. 📄 **Create entry point connecting server to stdio transport for Claude Desktop**
   `mcp/index.ts`

6. ✏️ **Add "mcp": "node dist/mcp/index.js" to scripts**
   `package.json`

7. 📄 **Register your MCP server in Claude Code settings**
   `.claude/mcp.json`
