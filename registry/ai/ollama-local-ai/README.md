# Local AI with Ollama

> Run Llama 3, Mistral, and Phi-3 locally — zero API cost, full privacy

## Why This Exists

API bills kill prototypes. Privacy requirements kill cloud AI. Ollama gives you production-quality models locally in 3 commands. When you're ready for prod, swap the provider — same code, same SDK.

## Steps (7 total)

1. ▶️ **Install Ollama (macOS) or download from ollama.com**
   `brew install ollama`

2. ▶️ **Download Llama 3.1 (8B model, ~5GB)**
   `ollama pull llama3.1`

3. ▶️ **Start Ollama server on localhost:11434**
   `ollama serve`

4. ▶️ **Install Vercel AI SDK with Ollama provider**
   `npm install ai @ai-sdk/ollama`

5. 📄 **Create localAI client pointing to Ollama with fallback to OpenAI when OLLAMA_AVAILABLE env is false**
   `lib/local-ai.ts`

6. ✏️ **Add OLLAMA_HOST=http://localhost:11434 and OLLAMA_MODEL=llama3.1**
   `.env.local`

7. 📄 **Update chat route to use localAI — zero code change needed**
   `app/api/chat/route.ts`
