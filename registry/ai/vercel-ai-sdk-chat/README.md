# AI Streaming Chat (Vercel AI SDK)

> Add a streaming GPT-4o chatbot to any Next.js app in under 30 minutes

## Why This Exists

Streaming is not optional — users abandon AI features that wait 5 seconds before responding. The Vercel AI SDK handles the stream protocol, error recovery, and React state so you don't have to invent it yourself.

## Steps (8 total)

1. 🔍 **Detect Next.js version and existing AI dependencies**
   `package.json`

2. ▶️ **Install Vercel AI SDK and OpenAI provider**
   `npm install ai @ai-sdk/openai`

3. 📄 **Create streaming POST /api/chat route using streamText() with message history support**
   `app/api/chat/route.ts`

4. 📄 **Create Chat component with useChat hook, message list, input form, and loading indicator**
   `components/Chat.tsx`

5. 📄 **Create Message component with role-based styling (user vs assistant), markdown rendering**
   `components/Message.tsx`

6. ✏️ **Add OPENAI_API_KEY environment variable**
   `.env.local`

7. ✏️ **Add Chat component to the main page**
   `app/page.tsx`

8. ▶️ **Start dev server and verify streaming works end to end**
   `npm run dev`
   *Expected: Chat streams responses*
