# Progressive Web App with Next.js

> Install your web app on any device — offline support, push notifications, home screen

## Why This Exists

A PWA is a native-quality app without an App Store. Users install it from the browser, it works offline, and you push updates instantly. For content-heavy or B2B apps, PWAs outperform both web and native in engagement metrics.

## Steps (6 total)

1. ▶️ **Install next-pwa (Workbox-powered, TypeScript compatible)**
   `npm install @ducanh2912/next-pwa`

2. ✏️ **Wrap next config with withPWA() — configure caching strategies and workbox options**
   `next.config.ts`

3. 📄 **Create Web App Manifest: name, icons (192px, 512px), theme_color, display: standalone, start_url**
   `public/manifest.json`

4. ✏️ **Add <link rel="manifest">, meta theme-color, and apple-touch-icon to document head**
   `app/layout.tsx`

5. 📄 **Create subscribeToPush(), sendPushNotification(), and unsubscribe() using Web Push API**
   `lib/push-notifications.ts`

6. 📄 **Add custom service worker logic: background sync, offline page, push event handler**
   `public/sw-custom.js`
