# n8n Workflow Automation

> No-code automation for non-tech users — connect 400+ apps without writing code

## Why This Exists

Non-technical users need to automate without asking developers. n8n is the self-hosted alternative to Zapier — unlimited workflows, no per-task pricing, and your data never leaves your servers. One deployment saves thousands in Zapier costs.

## Steps (5 total)

1. ▶️ **Start n8n with persistent storage**
   `docker run -d --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n n8nio/n8n`

2. 📄 **Create production n8n compose with PostgreSQL backend, SMTP config, and webhook URL**
   `n8n/docker-compose.yml`

3. 📄 **Create webhook endpoint that n8n can trigger to start workflows from your app events**
   `app/api/webhooks/n8n/route.ts`

4. 📄 **Create triggerWorkflow(workflowId, data) function using n8n REST API for app-initiated automations**
   `lib/n8n.ts`

5. ✏️ **Add N8N_BASE_URL, N8N_API_KEY, N8N_WEBHOOK_URL**
   `.env.local`
