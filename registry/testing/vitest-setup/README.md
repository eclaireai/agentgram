# Vitest Testing Framework

> Blazing fast unit tests with TypeScript, coverage, and watch mode

## What it does

Set up Vitest from scratch: TypeScript support, V8 coverage reports, path aliases, and a sample test. Runs 10x faster than Jest on cold starts.

## Why this recipe exists

Jest requires Babel transforms for TypeScript. Vitest uses Vite's native ESM pipeline — no config, instant startup, identical API. If you're using Vite or just want fast tests, this is the obvious choice.
