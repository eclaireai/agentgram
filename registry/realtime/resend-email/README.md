# Transactional Email with Resend

> Beautiful HTML emails with React Email — and actually get delivered to inbox

## Why This Exists

Sendgrid is expensive and over-engineered for transactional email. Resend is built for developers — send with one API call, design with React, track opens and clicks, and get 3000 emails/month free.

## Steps (6 total)

1. ▶️ **Install Resend and React Email**
   `npm install resend @react-email/components @react-email/render`

2. 📄 **Create welcome email template with company branding, user name, and CTA button**
   `emails/WelcomeEmail.tsx`

3. 📄 **Create password reset email with secure reset link and 24-hour expiry notice**
   `emails/PasswordResetEmail.tsx`

4. 📄 **Create invoice email with line items table, total, and PDF attachment support**
   `emails/InvoiceEmail.tsx`

5. 📄 **Create sendEmail(to, template, props) wrapper with error handling and dev-mode preview**
   `lib/email.ts`

6. ✏️ **Add RESEND_API_KEY and RESEND_FROM_EMAIL (e.g. noreply@yourdomain.com)**
   `.env.local`
