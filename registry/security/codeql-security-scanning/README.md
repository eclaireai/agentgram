# CodeQL Security Scanning + Dependabot

> Catch vulnerabilities in your code and dependencies automatically — before they ship

## Why This Exists

Security tools run by humans are run inconsistently. CodeQL runs on every PR and finds vulnerabilities that human review misses. Dependabot catches the Log4Shell-style supply chain attacks automatically. Both are free for public repos.

## Steps (5 total)

1. 📄 **Create CodeQL analysis workflow on push/PR: javascript and typescript languages**
   `.github/workflows/codeql.yml`

2. 📄 **Configure Dependabot: npm updates weekly, GitHub Actions updates monthly**
   `.github/dependabot.yml`

3. 📄 **Add npm audit --audit-level=high to CI — fail build on high-severity vulnerabilities**
   `.github/workflows/security.yml`

4. 📄 **Create security policy: responsible disclosure, contact email, expected response time**
   `.github/SECURITY.md`

5. ▶️ **Run initial audit to baseline current vulnerability state**
   `npm audit`
   *Expected: N vulnerabilities*
