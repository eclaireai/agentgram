# Environment Variable Validation

> Crash at startup if required env vars are missing — not at 2am in production

## What it does

Validate all environment variables at startup using Zod. If any required variable is missing or has the wrong type, the app refuses to start with a clear error message listing exactly what's wrong.

## Why this recipe exists

Silent missing env vars are the #1 cause of confusing production incidents. This recipe makes your app fail loudly at startup — before it serves a single request — if the configuration is wrong.
