# Clerk Auth for Next.js

> Complete auth with social login, MFA, and user management in 10 minutes

## Why This Exists

Building auth from scratch takes weeks and you'll get it wrong. Clerk handles OAuth, MFA, session management, bot detection, and compliance. The 10 minutes you spend here saves 3 sprints.

## Steps (8 total)

1. ▶️ **Install Clerk Next.js SDK**
   `npm install @clerk/nextjs`

2. ✏️ **Add NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY and CLERK_SECRET_KEY from Clerk dashboard**
   `.env.local`

3. 📄 **Create Clerk middleware with clerkMiddleware() and route protection rules**
   `middleware.ts`

4. ✏️ **Wrap root layout with <ClerkProvider> for global auth context**
   `app/layout.tsx`

5. 📄 **Add Clerk <SignIn> component with custom redirect URL**
   `app/(auth)/sign-in/[[...sign-in]]/page.tsx`

6. 📄 **Add Clerk <SignUp> component**
   `app/(auth)/sign-up/[[...sign-up]]/page.tsx`

7. 📄 **Create webhook handler for user.created, user.updated, user.deleted events — sync to your DB**
   `app/api/webhooks/clerk/route.ts`

8. 📄 **Create getUser() helper for server components and currentUser() for API routes**
   `lib/auth.ts`
