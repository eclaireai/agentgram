# Auth.js v5 (Next-Auth)

> Framework-agnostic auth with 50+ OAuth providers — the open-source standard

## Why This Exists

Auth.js is the most battle-tested open-source auth library for JavaScript. No vendor lock-in, self-hostable, works with any database via adapters. If you want full control over your auth without building from scratch, this is it.

## Steps (7 total)

1. ▶️ **Install Auth.js v5 beta and Drizzle adapter**
   `npm install next-auth@beta @auth/drizzle-adapter`

2. 📄 **Create Auth.js config with providers, Drizzle adapter, callbacks for JWT and session**
   `auth.ts`

3. 📄 **Create edge-compatible config with authorized() callback for middleware use**
   `auth.config.ts`

4. 📄 **Create auth middleware using auth.config.ts (edge-safe)**
   `middleware.ts`

5. 📄 **Create Auth.js Drizzle schema: users, accounts, sessions, verificationTokens tables**
   `db/schema/auth.ts`

6. ✏️ **Add AUTH_SECRET, GITHUB_ID, GITHUB_SECRET, GOOGLE_ID, GOOGLE_SECRET**
   `.env.local`

7. 📄 **Create catch-all Auth.js route handler**
   `app/api/auth/[...nextauth]/route.ts`
