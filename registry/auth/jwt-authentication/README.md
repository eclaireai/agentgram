# JWT Authentication

> Stateless auth with access + refresh tokens and bcrypt password hashing

## What it does

Add production-ready JWT authentication to any Express/Node.js API. Includes access tokens (15-min expiry), refresh tokens (7-day expiry), bcrypt password hashing, auth middleware, and a full test suite.

## Why this recipe exists

Session auth breaks in distributed systems. JWTs are stateless, scalable, and work across microservices. This recipe gives you the full pattern — not just the happy path.
