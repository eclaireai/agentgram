# Full-Text Search with Meilisearch

> Instant search with typo tolerance in <50ms — self-hosted or cloud

## Why This Exists

SQL LIKE queries are slow and have no typo tolerance. Elasticsearch requires dedicated DevOps. Meilisearch delivers sub-50ms results with typo correction and relevance ranking out of the box, with a 5-minute setup.

## Steps (7 total)

1. ▶️ **Start Meilisearch for local development**
   `docker run -d -p 7700:7700 getmeili/meilisearch:latest`

2. ▶️ **Install Meilisearch client and React InstantSearch**
   `npm install meilisearch react-instantsearch`

3. 📄 **Create Meilisearch client, createIndex(), indexDocuments(), and search() helper**
   `lib/search.ts`

4. 📄 **Create indexing script that fetches records from DB and sends to Meilisearch**
   `scripts/index-data.ts`

5. 📄 **Create search UI with InstantSearch, hits, pagination, and facet filters**
   `components/SearchBox.tsx`

6. 📄 **Create server-side search proxy with authentication and result shaping**
   `app/api/search/route.ts`

7. ▶️ **Index existing data into Meilisearch**
   `npx tsx scripts/index-data.ts`
   *Expected: N documents indexed*
