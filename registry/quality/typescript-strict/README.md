# TypeScript Strict Mode

> Enable strictest TypeScript settings and fix every resulting error

## What it does

Upgrade a TypeScript project to strict mode: enable strict, noUncheckedIndexedAccess, exactOptionalPropertyTypes, and fix every resulting type error. Leaves the codebase bulletproof.

## Why this recipe exists

TypeScript without strict is like a seatbelt you never buckle. strict catches null dereferences, implicit any, and missing return types. This recipe enables it and fixes every error it reveals — usually 20-50 real bugs.
