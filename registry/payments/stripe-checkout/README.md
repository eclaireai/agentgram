# Stripe Checkout

> Accept payments in 30 minutes — one-time, saved cards, Apple Pay included

## Why This Exists

Stripe Checkout offloads PCI compliance, card validation, 3D Secure, and Apple/Google Pay to Stripe. You write 50 lines, not 500. Every hour you spend on custom payment UI is an hour not building your product.

## Steps (7 total)

1. ▶️ **Install Stripe Node.js SDK**
   `npm install stripe`

2. 📄 **Create Stripe client singleton with API version pinned to latest stable**
   `lib/stripe.ts`

3. 📄 **Create POST /api/checkout that creates Stripe Checkout Session with success_url and cancel_url**
   `app/api/checkout/route.ts`

4. 📄 **Create success page that retrieves session and shows order confirmation**
   `app/checkout/success/page.tsx`

5. 📄 **Create Stripe webhook handler with signature verification, handle payment_intent.succeeded and checkout.session.completed**
   `app/api/webhooks/stripe/route.ts`

6. ✏️ **Add STRIPE_SECRET_KEY, STRIPE_WEBHOOK_SECRET, NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY**
   `.env.local`

7. ▶️ **Test webhooks locally with Stripe CLI**
   `stripe listen --forward-to localhost:3000/api/webhooks/stripe`
