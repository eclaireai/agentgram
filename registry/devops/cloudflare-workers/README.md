# Cloudflare Workers + D1

> Deploy serverless API to 300+ edge locations with SQLite at the edge

## Why This Exists

Traditional serverless (Lambda) has cold starts and runs in one region. Cloudflare Workers have no cold start and run in 300+ cities worldwide. For global read-heavy APIs, this is 10x faster than a regional server.
