# Vercel Deployment

> Zero-config deploy to Vercel with preview URLs, environment variables, and edge config

## What it does

Configure a Next.js or Node.js project for Vercel deployment. Includes vercel.json, environment variable setup in Vercel dashboard, preview URL configuration, and GitHub integration for automatic deploys.

## Why this recipe exists

Vercel's preview URLs are a superpower — every PR gets a live URL. But the initial setup (env vars, rewrites, edge config) is easy to get wrong. This recipe does it once, perfectly.

## Steps (5 total)

1. 🔍 **`find`** → `package.json`  
   Check framework and build configuration

2. 📄 **`create_file`** → `vercel.json`  
   Create vercel.json with framework, buildCommand, outputDirectory, and rewrites

3. ✏️ **`modify_file`** → `.env.example`  
   Document all required environment variables for Vercel dashboard setup

4. 📄 **`create_file`** → `.vercelignore`  
   Add tests, seeds, docs, and scripts to Vercel ignore

5. ⚡ **`run_command`** → `npx vercel --prod --confirm`  
   Deploy to Vercel production and verify URL
