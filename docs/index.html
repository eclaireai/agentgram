Get started

Up and running in 30 seconds.

Works as a global CLI, a zero-install npx command, or an MCP server wired directly into Claude Code. 

# Install globally
npm install -g agentgram

# Auto-record all Claude Code sessions
agentgram hook install

# Check any task before you start
agentgram preflight "your task"
# No install required — just run
npx agentgram preflight "add stripe webhooks to nextjs"
// .claude/.mcp.json
{
  "mcpServers": {
    "agentgram": {
      "command": "npx",
      "args": ["agentgram", "mcp"]
    }
  }
}
